# Retail Sales Analytics Platform

## Overview
Built an end-to-end analytics pipeline using Python, PostgreSQL, SQL, and Tableau to analyze 51K+ retail transactions.

## Tech Stack
- Python (Pandas, NumPy)
- PostgreSQL
- SQL
- Jupyter Notebook
- Tableau Public

## Key Insights
- Total Revenue: $12.64M
- Technology generated highest revenue ($4.74M)
- Top customer generated ~$40K revenue
- Revenue peaked in November–December
